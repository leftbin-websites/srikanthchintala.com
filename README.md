# Srikanth Chintala - Personal Blog

Personal blog and professional website for Srikanth Chintala - High Court Lawyer, Documentary Filmmaker, Ambedkarite, and Humanist.

## About

This website serves as a platform for constitutional law insights, human rights advocacy, and legal commentary.

## Development

### Running locally

```bash
make develop
# or
npm install
npm run dev
```

Visit http://localhost:5173

### Building for production

```bash
make build
# or
npm run build
```

### Preview production build

```bash
make preview
# or
npm run build && npm run preview
```

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `main` branch.

**Live site:** https://srikanthchintala.com

## Tech Stack

- React + Vite
- Tailwind CSS
- Content management system
- GitHub Pages for hosting
