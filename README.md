# Srikanth Chintala - Personal Website

This is the personal website of Srikanth Chintala - High Court Lawyer, Documentary Filmmaker, Ambedkarite, and Humanist.

Built with Vite + React.

## Running the app

```bash
npm install
npm run dev
```

## Building the app

```bash
npm run build
```

## Preview the build

```bash
npm run preview
# or
make preview
```
